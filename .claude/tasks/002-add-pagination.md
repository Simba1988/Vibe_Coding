# Task 002: Add Pagination

## Goal

Add Pagination to movie list

## Requirements

- Only show 10 movies per page, 5 per line
- Have arrows on the bottom to navigate between pages
- Show current page number
- Show total number of pages
- Show "No results" message when no movies are found

## Constraints

- Angular 21.2.0 standalone components
- Use HttpClient
- Use RxJS operators, not imperative event handling
- Keep styling simple

## Acceptance criteria

- Search only shows 10 results per page
- Pagination arrows are visible
- Current page number is visible
- Total number of pages is visible
- "No results" message is visible when no movies are found
